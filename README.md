# Tự học Next.js (FE)

Chào mừng bạn đến với kho lưu trữ (repository) cá nhân của tôi, nơi tôi ghi chép và thực hành các kiến thức về **Next.js**.

## Mục tiêu
- Nắm vững kiến thức từ cơ bản đến nâng cao của Next.js (App Router).
- Thực hành xây dựng giao diện hiện đại, tối ưu hiệu năng và SEO.
- Áp dụng các kỹ thuật như Server Components, Client Components, và Data Fetching.

Dưới đây là danh sách các nội dung tôi đã và đang tìm hiểu:

1.  **[Giới thiệu Next.js là gì?](./1.Nextjs-la-gi.md)** - Tổng quan về framework, lý do chọn Next.js so với React thuần.
2.  **Cấu trúc thư mục** - Tìm hiểu cách tổ chức file trong App Router (Đang cập nhật).
3.  **Routing & Navigation** - Cách tạo page, layout và chuyển trang (Đang cập nhật).

## 🛠 Công nghệ sử dụng
- **Framework:** Next.js 15+ (App Router)
- **Language:** JavaScript / TypeScript
- **Styling:** Tailwind CSS
